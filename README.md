## Práctica 1: Repaso de clases y herencia <br>

**Elaboró:** Lic. Erik Quintero Villeda <br>

---

### 1. Objetivos <br>
Repasar los conceptos de **herencia** en Java y la definición de clases. <br>

### 2. Instrucciones <br>
Implementa los métodos faltantes de las clases: `PoligonoRegular`, `TrianguloEquilatero`, `Pentagono`, `Cuadrado` y `Circulo`. <br>

1. **`calcularPerimetro` (PoligonoRegular)** <br>
Método que calcula el perímetro de un polígono regular. <br>

2. **`calcularPerimetro` (Circulo)** <br>
Método que calcula el perímetro de un círculo. <br>

3. **`calcularArea` (TrianguloEquilatero, Pentagono, Cuadrado y Circulo)** <br>
Método que calcula el área de la figura correspondiente. <br>

---

### Ejecución <br>
Se utilizará una clase llamada `MainFiguras`, la cual contiene un método `main` que servirá para probar cada uno de los métodos que implementaste (las pruebas se imprimirán en la terminal). <br>
